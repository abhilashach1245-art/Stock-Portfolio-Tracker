# 📈 Stock Portfolio Tracker (Python)

A simple Stock Portfolio Tracker built using Python that helps users manage and track their stock investments. This project allows users to add stocks, view their portfolio, and calculate total investment value.

## 📌 Features

* Add stocks with quantity and price
* View current portfolio
* Calculate total investment value
* Simple and user-friendly interface
* Command-line based application

## 🛠️ Technologies Used

* Python
* Basic data structures (lists, dictionaries)

## 🎯 How It Works

* The program stores stock details using Python data structures.
* Users can input stock name, quantity, and price.
* It calculates total investment by summing all stock values.
* Displays portfolio details on demand.
